#4 Ontwerphandleiding - 2019

Halverwege jaar 1 kregen we de opdracht om een klein simpel ontwerphandleiding te maken. Hier moesten we de gebruikte onderzoeksmethodes beschrijven en uitleggen. Uiteindelijk ben ik best blij met de vormgeving en leesbaarheid van dit boekje. Per onderzoeksmethode is er duidelijk beschreven hoe je deze uitvoert en wanneer je deze kan gebruiken.

#3 Reisgids - 2019

In het tweede jaar kregen we de opdracht om een reisgids te ontwerpen. Het was de bedoeling dat deze in Mavel, AdobeXD of een ander soortgelijke programma werd ontworpen. Ik wou het zelf graag in HTML en CSS doen. Dit heb ik dan ook gedaan. Helaas kreeg ik onvoldoende tijd voor waardoor mijn code niet volledig symantisch correct is. De focus lag hier op wat laat ik wanneer zien. Wat en hoe worden elementen weergegeven op mobiel en wat op PC. Aan de hand van display block en none heb ik bepaalde elementen zichtbaar of verborgen gezet op basis van het scherm grootte. Daarnaast is de content zo veel mogelijk responsive ontworpen met behulp van grid en flexbox. Ondanks dat de symantiek niet 100% correct is, ben ik erg trots op mijn eindresultaat. Voornamelijk aangezien ik dit nog in een vrij korte tijdspanne in elkaar heb kunnen zetten.

Dit heb ik altijd al willen maken. Al mag het misschien wat simpel zijn, en zou ik het graag nog verder willen uitbreiden, ben ik alsnog erg trots op dit werk. In dit simpele textbased RPG ga je op avontuur en vecht je tegen verschillende vijanden die elke keer andere statistieken hebben.

#1 Hyrule Shield - 2019

Voor een extra studiepunt kreeg ik de taak om een afbeelding na te maken in CSS. Nu ben ik een grote gamer en koos ik voor een schild uit de game Legend of Zelda. Ik ben erg trots op het uiteindelijke resultaat. Ik ben vooral trots op de vleugels van de vogel op het schild. Die waren lastig om te maken.
