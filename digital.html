Reisgids - 2019

In het tweede jaar kregen we de opdracht om een reisgids te ontwerpen. Het was de bedoeling dat deze in Mavel, AdobeXD of een ander soortgelijke programma werd ontworpen. Ik wou het zelf graag in HTML en CSS doen. Dit heb ik dan ook gedaan. Helaas kreeg ik onvoldoende tijd voor waardoor mijn code niet volledig symantisch correct is. De focus lag hier op wat laat ik wanneer zien. Wat en hoe worden elementen weergegeven op mobiel en wat op PC. Aan de hand van display block en none heb ik bepaalde elementen zichtbaar of verborgen gezet op basis van het scherm grootte. Daarnaast is de content zo veel mogelijk responsive ontworpen met behulp van grid en flexbox.
